Selector Types

This is how we can 'select' the various elements on our page

Classes

In CSS, a class is a reusable styling rule that can be applied to multiple elements on a webpage. Classes help organize and maintain styles efficiently, allowing for consistent and scalable design across a website.

A CSS class is defined using a dot (.) prefix in a CSS file and applied to elements using the class attribute in HTML.

IDs

In CSS, an ID selector is used to apply styles to a specific, unique element on a webpage. Unlike classes, which can be reused across multiple elements, an ID should only be assigned to one element per page for best practices.

Link Selector

The link selector in CSS is used to style hyperlinks in an HTML document. It allows you to define how links appear in different states, such as when they are unvisited, visited, hovered over, or active.
